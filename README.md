# honoc

A CLI command for Hono

## Installation

```bash
npm install -g honoc
```

## Usage

```bash
# Show help
honoc --help

# Say hello
honoc hello

# Say hello to someone
honoc hello yusuke
```

## Commands

- `hello [name]` - Say hello (default: Hono)
