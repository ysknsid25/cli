# @hono/cli

CLI for Hono

## Installation

```bash
npm install -g @hono/cli
```

## Usage

```bash
# Show help
hono --help

# Say hello
hono hello

# Say hello to someone
hono hello yusuke

# Open documentation in browser
hono docs
```

## Commands

- `hello [name]` - Say hello (default: Hono)
- `docs` - Open Hono documentation in browser
